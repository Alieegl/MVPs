# Hoja de Ruta (Roadmap)

## Fase 1: MVP - Base y Visualización (Completado)
- [x] Arquitectura base React + Tailwind.
- [x] Dashboard con KPIs básicos.
- [x] **Navegación Móvil:** Menú tipo Drawer.
- [x] **Optimización Móvil:** Vistas de tarjetas.

## Fase 2: Interacción y Flujos (Completado)
- [x] **Módulo de Carga:** Formulario para subir nuevos documentos.
- [x] **Filtros Avanzados:** Buscador global y filtros por estado.
- [x] **Gestión de Rechazos:** Modal con comentario obligatorio y control de versiones.
- [x] **Estructura UCO:** Carpetas (Planeación, Ejecución, etc.) e Integración PowerBI.
- [x] **Feedback:** Notificaciones (Toasts) y Tooltips de tiempo.

## Fase 3: Persistencia y Autenticación (Próximos Pasos)
- [ ] Integración con Backend (Node.js/Firebase).
- [ ] Autenticación real (Microsoft 365).
- [ ] Carga real de archivos (Blob Storage).

## Fase 4: Inteligencia Avanzada
- [ ] Análisis predictivo de retrasos.
- [ ] Chatbot con búsqueda profunda sobre contenido de PDFs.
- [ ] Generación automática de actas.
